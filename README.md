## 藥局端頁面(/Pharmacy)

![](/pics/藥局_1.jpg)


從首頁最下方登入藥局頁面

![](/pics/藥局_3.jpg)


新增藥物

![](/pics/藥局_2.jpg)


查詢藥物

![](/pics/藥局_4.jpg)


所有患者開藥紀錄顯示於頁面最下方, 分為未領(開藥紀錄)和已領藥

![](/pics/藥局_5.jpg)


若患者欲領藥, 點擊未領紀錄最後方按鈕「患者領取」進行領藥

![](/pics/藥局_6.jpg)


患者領藥後, 紀錄會出現在已領藥區域, 並新增領藥時間

![](/pics/藥局_7.jpg)
